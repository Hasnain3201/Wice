# WICE Frontend

This directory contains the React/Vite application for the WICE platform.

For the full project overview, screenshots, setup steps, and security guidance, see the root [README.md](../README.md).

## Local Commands

```bash
npm install
npm run dev
npm run lint
npm run build
npm run preview
```

## Environment

Copy the example file and fill values locally:

```bash
cp .env.example .env.local
```

Never commit `.env.local` or account passwords.
